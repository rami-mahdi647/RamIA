# RamIA MVP Flow

This is the v1 happy path for a new user. The steps below are written exactly in product order and should be reflected in onboarding copy.

## 6-step happy path

1. **install**  
   User installs RamIA and completes first-run setup.

2. **open dashboard**  
   User opens the RamIA dashboard and sees wallet status and action buttons.

3. **create wallet+backup**  
   User creates a wallet and confirms backup instructions before continuing.

4. **get coins**  
   User receives starter/test coins so they can try real actions.

5. **send with guardian warning**  
   User starts a send, sees AI Guardian risk guidance, and then confirms or cancels.

6. **optional Stripe bot rental**  
   User can optionally start a Stripe-powered bot rental flow.

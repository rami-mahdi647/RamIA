RamIA

This website is a static launcher + documentation. The node runs locally (Termux/PC).

Download

  • GitHub Repository
  • Download ZIP

Run in Termux (quick)

pkg update -y && pkg upgrade -y
pkg install -y python git
git clone https://github.com/rami-mahdi647/RamIA.git
cd RamIA
# train guardian (demo)
python3 aiguardian.py train --csv dataset.csv --out guardian_model.json
# run node locally
python3 aicore.py --guardian-model guardian_model.json node --web
# open: http://127.0.0.1:8787
